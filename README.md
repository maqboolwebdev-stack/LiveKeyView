    1. Key Visualizer

    This project is a simple JavaScript application that shows the key pressed on the keyboard directly on the screen. It updates the text with the pressed key, changes the text color randomly, and also changes the background color randomly. The Space key is handled separately to display the word Space instead of a blank.

    2. Features
    Key press detection in real time  
    Random text color change on each key press  
    Random background color change on each key press  
    Special handling for the Space key  
    Smooth animation effect when a key is pressed  

    3. How to Use
    Open the project in a browser  
    Press any key on your keyboard  
    Watch the key name appear in the center with animation and color changes  

    4. Technologies
    HTML  
    CSS  
    JavaScript  

    5. Future Improvements
